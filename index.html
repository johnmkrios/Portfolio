FindIt

Description

FindIt is a mobile app created using JavaScript, HMTL and CSS, that allows users to browse through screenshots from the Lighthouse.

Functionality

  • Generating different screenshots using URL patterns.
  • Invalid URLs will be disregarded automatically
  • Only the screenshots will be displayed after the page is done loading. Other elements will be disregarded.
  • Users are able to retrieve old screenshots by going back.
  • Users have the option to copy and share the URLs, or open them in Web browser.

J-Mak Flights

Description

J-Mak Flights is a data management software created using C#, WPF and XML data binding. This software acts as a platform for all users to access and edit data. J-Mak Flights is designed to dynamically populate parameters with data from a real database such as MySQL or MongoDB.

Functionality

  • Different account types with extra privileges granted to the Super account type.
  • Privileged users are able to add, update or delete data.
  • Regular accounts can be deleted, added and managed by the Super accounts.
  • Context menus accessibility for easier controls and a user-friendly platform.
